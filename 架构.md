# 家校通架构

## 服务端

服务端使用`java`语言

1. 框架使用`SSH` 
2. `REST` 使用 Spring 实现

**日后将 Struts2 替换为 Spring MVC**

## 客户端

客户端使用 Android

1. `ORM `使用 `ormlite`
2. `REST` 使用 ~~Retrofit~~ `Spring for android`

支持的版本暂定为 `4.0+`

## 通信

1. 客户端与服务端之间通信使用 `json`
2. ~~客户端与服务器都使用 `JSON-lib`~~
3. 使用 RESTFUL

## REST WEB
1. 显示的使用 HTTP 方法
2. 无状态
3. 公开目录结构式的 URI
4. 传输 XML、JavaScript Object Notation(JSON), 或者同时这两种

**使用 Spring MVC 实现 REST**
